  # PDF:
  * PDF stands for Portable Document Format
  * Developed by Adobe in 1990s
  * It includes text, graphics and images independent of software
  * Same format in Apple MAC or Microsoft word
  
  # General structure of PDF file:
  * Header
  * Body
  * Xref table
  * Trailer
